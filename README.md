# 🎮 GameArena

My sixth assignment during the Front-End Development track at Route Academy.

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Font Awesome

## Features
- Responsive Design
- Fixed Navbar
- Hero Section
- Our Games
- What We Do
- Scrolling Logos
- Team Section
- Contact Form
- Footer
